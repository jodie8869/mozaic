Mozaic

Transform a canvas picture into a photomosaic of small thumbnail images

Mozaic is a jQuery based JavaScript library for transforming an HTML5 canvas into a photomosaic of small images.

The underlying data structure used to find the closest tile image is a kd-tree with k = 3. This library is adapted from the Data Structures and Algorithms course from the University of Illinois at Urbana-Champaign.
